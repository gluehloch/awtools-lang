Das Package Behandelt die Verabeitung von Übersetzungen. Translator beschäftigt sich mit der Übersetzung von Sprach-Schlüsseln. Die getString(String, Object...) Methode ersetzt die Platzhalter {0} bzw. {1} ... {N} durch die entsprechende Ersetzung. Der Index der Platzhalter entspricht dem Index der übergebenen Parameterliste.
